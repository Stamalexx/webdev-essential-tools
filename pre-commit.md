# Pre-commit

Pre-commit is a script that runs **ESLint** and **Prettier** checks on our code when we run:

```bash
git commit
```

## What you NEED to do:

Install it by running(requires python and pip):

```bash
pip install pre-commit
```

Check if it was installed by running:

```bash
pre-commit --version
```
